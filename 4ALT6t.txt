<p class='' id=''>The read-only localStorage property allows you to access a Storage object for the Document's origin; the stored data is saved across browser sessions. localStorage is similar to sessionStorage, except that while data stored in localStorage has no expiration time, data stored in sessionStorage gets cleared when the page session ends â€” that is, when the page is closed.</p>
<br />
<p class='' id=''>
	<p class='' id=''>It should be noted that data stored in either
		<p class='badge badge-primary' id=''>localStorage</p> or
		<p class='badge badge-primary' id=''>sessionStorage</p>is specific to the protocol of the page.</p>
	<br /></p>

<p class='' id=''>The keys and the values are always strings (note that, as with objects, integer keys will be automatically converted to strings).</p>

<b>Syntax</b>
<p class='alert alert-dark' id=''>myStorage = window.localStorage;</p>
<br />
<b>Value</b>
<p class='' id=''>A Storage object which can be used to access the current origin's local storage space.</p>
<br />
<b>Exceptions</b>
<p class='badge badge-warning' id=''>SecurityError</p>
<br />
<p class='' id=''>The request violates a policy decision, or the origin is not a valid scheme/host/port tuple (this can happen if the origin uses the file: or data: scheme, for example). For example, the user may have their browser configured to deny permission to persist data for the specified origin.</p>
<br />
<h5 class='' id=''>Example</h5>
<br />
<p class='' id=''>The following snippet accesses the current domain's local Storage object and adds a data item to it using</p>
<br />
<p class='alert alert-dark' id=''>localStorage.setItem('myCat', 'Tom');</p>
<br />
<p class='' id=''>The syntax for reading the localStorage item is as follows:</p>
<br />
<p class='alert alert-dark' id=''>var cat = localStorage.getItem('myCat');</p>
<br />
<p class='' id=''>The syntax for removing the localStorage item is as follows:</p>
<br />
<p class='alert alert-dark' id=''>localStorage.removeItem('myCat');</p>
<br />
<p class='' id=''>The syntax for removing all the localStorage items is as follows:</p>
<br />
<p class='alert alert-dark' id=''>localStorage.clear(); // Clear all items</p>
