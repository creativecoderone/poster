<h5 class='' id=''>React</h5>
<br />
<p class='badge badge-warning' id=''>
	<a href='https://reactjs.org/' class='center' id=''>React</a>
</p>
<br />
<p class='' id=''>React is a javascript library developed by Facebook which provides very fast performance due to its virtual DOM. It can be easily plugged into existing projects. React has a lot of potential because of its active community which constantly develops its components. One of its coolest features is its ability to update only the part that needs to be changed once the “Refresh” button has been hit.</p>
